# Interactive Features: Функциональные возможности

Институциональный портал PSB Bank оснащен набором специализированных модулей для работы с финансовой информацией. Все функции адаптированы под концепцию «Swiss Elite UI».

## 1. Система публичных реестров (Registry Engine)
Основной функциональной единицей портала является **RegistryGrid** — унифицированная система таблиц для вывода больших массивов данных из PocketBase.

- **Реализованные реестры**:
  - Банки-партнеры (Partner Banks).
  - Валютные бюро (Forex Bureaus).
- **Возможности**:
  - **Мгновенный поиск**: Фильтрация записей «на лету» по названию, стране или статусу.
  - **Статус-валидация**: Визуальная индикация активных лицензий и верифицированных участников.
  - **Адаптивность**: Таблицы автоматически перестраиваются под мобильные устройства без потери заголовков.

## 2. Ин институциональная навигация (Breadcrumbs)
Система реализована через специализированный компонент, который:
- Автоматически выстраивает логический путь пользователя.
- Поддерживает два цветовых режима (Navy/White) для разных фоновых секций.
- Обеспечивает мгновенный возврат в родительские разделы (Institutional, Banking & Markets и др.).

## 3. Динамический рендеринг контента (Generic Engine)
Для текстовых страниц, содержащих большие объемы нормативной информации (Licensing, Regulations), используется движок рендеринга на базе Tailwind Typography:
- **Премиальная верстка**: Автоматическое применение стилей «прозы» к HTML-контенту из базы/JSON.
- **Поддержка иерархии**: Корректное отображение списков, вложенных заголовков и цитат в институциональном стиле.

## 4. Профили руководителей (Executive Modals)
В разделе **Board of Directors** внедрена система модальных окон для детального ознакомления с биографиями руководства:
- **Анимация**: Плавные переходы через Framer Motion.
- **Инфо-карты**: Структурированное представление профессионального опыта без перезагрузки страницы.

## 5. Система безопасности «Quantum Shield»
Визуально-технический слой, обеспечивающий защиту институционального контента:
- **Контроль копирования**: Отключение контекстного меню и стандартного выделения текста в критических зонах.
- **Anti-Scraping**: Защита от автоматизированного сбора данных через блокировку определенных паттернов поведения.
- **Визуальные маркеры подлинности**: Динамические печати и водяные знаки, подтверждающие актуальность документов.

## 6. Адаптивность и производительность
- **Mobile First**: 100% корректное отображение на смартфонах и планшетах.
- **Zero CLS**: Оптимизация шрифтов и ассетов для предотвращения скачков контента при загрузке.
- **SEO Ready**: Семантическая разметка для корректного индексирования поисковыми системами.
