# Overview: Pacific Settlement Bank (PSB) Portal

## Описание проекта
**PSB Portal** — это высокотехнологичный институциональный веб-портал, разработанный для Pacific Settlement Bank. Проект представляет собой современную цифровую витрину банка в эстетике «Swiss Conservative» (швейцарский консерватизм), сочетающую строгий финансовый дизайн с передовыми технологиями безопасности.

Портал предназначен для публикации официальных реестров, нормативных документов, новостей и институциональной информации банка.

## Технологический Стек
Проект построен на современном и производительном стеке:

- **Frontend**: [Next.js 15](https://nextjs.org/) (App Router) — высокопроизводительный React-фреймворк с поддержкой серверного рендеринга (SSR) и статической генерации (SSG).
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/) — современный CSS-фреймворк для быстрой и гибкой верстки с использованием современных CSS-переменных.
- **Backend / CMS**: [PocketBase](https://pocketbase.io/) — легковесная и быстрая база данных реального времени, выступающая в роли CMS для управления реестрами.
- **Animations**: [Framer Motion](https://www.framer.com/motion/) — библиотека для создания плавных институциональных переходов и микро-взаимодействий.
- **Language**: TypeScript — для строгой типизации и надежности кодовой базы.

## Архитектура решения
Архитектура портала строится на гибридном подходе к управлению данными:

1.  **Static Layer**: Основные институциональные страницы («О банке», «Функции» и т.д.) используют статический контент из JSON-файлов, что обеспечивает мгновенную загрузку.
2.  **Dynamic Registry Layer**: Реестры банков-партнеров и валютных бюро получают данные напрямую из PocketBase API. Это позволяет администратору обновлять списки через удобную панель управления без вмешательства в код.
3.  **Security Layer (Quantum Shield)**: Глобальный провайдер безопасности, который защищает контент от несанкционированного копирования и предоставляет визуальные эффекты «слоя подлинности».
