# PSB Bank: Institutional Documentation Hub

Добро пожаловать в центр документации портала **Pacific Settlement Bank**. Этот каталог содержит полную информацию о технической архитектуре, функциональных возможностях и процессах управления контентом проекта.

## Карта документации

Для удобной навигации выберите интересующий вас раздел:

- [**Overview**](./overview.md) — Общее описание проекта, миссия, технологический стек и базовая архитектура.
- [**Features**](./features.md) — Каталог реализованных возможностей: реестры, система безопасности, динамические страницы и модальные окна.
- [**Project Structure**](./structure.md) — Техническая карта директорий. Описание того, как организован код и где хранятся данные.
- [**Deployment Guide**](./deployment.md) — Инструкция по развертыванию среды разработки, запуску фронтенда и сервера базы данных.
- [**Admin Guide**](./admin.md) — Руководство по управлению контентом через панель PocketBase для администраторов и редакторов.

## Быстрый старт (Разработчику)

Проект требует установленного Node.js 18+. Основной цикл запуска:

1.  Установка зависимостей: `npm install`
2.  Запуск портала: `npm run dev` (порт 3000)
3.  Запуск базы данных: `.\pb\pocketbase.exe serve` (порт 8090)
