# ПРОЕКТ PSB: ВНУТРЕННИЙ КОНТЕКСТ V2
(КОНФИДЕНЦИАЛЬНО — ОБНОВЛЕНИЕ С УЧЕТОМ АДМИНКИ)

## Обзор проекта (Уточненный)
- **Название:** Портал Pacific Settlement Bank (PSB)
- **Клиент:** Malina / Менеджер (FL.ru)
- **Бюджет:** 28 000 руб.
- **Главные Цели:** 
    1. «Дорогой», консервативный швейцарский дизайн (Navy, Gold, White).
    2. Админка для реестров (добавление строк за минуту).
    3. 52 страницы контента + защита Quantum Shield.

## Технологический стек V2
- **Frontend:** Next.js (App Router) + Tailwind CSS + Framer Motion (минимум анимаций, только плавность).
- **Backend/CMS:** PocketBase (админка из коробки).
- **Безопасность:** Кастомный модуль Quantum Shield.

## Дизайн-код (Swiss Bank Style)
- **Цвета:** Глубокий синий `#002147`, Золото `#C5A059`, Белый `#FFFFFF`.
- **Шрифты:** Премиальные гротески (Inter, Outfit или похожие).
- **Сетка:** Строгая, четкие границы, много «воздуха».

## Реестры и Админка
- В PocketBase создаем коллекции для банков и брокеров.
- Клиент сможет заходить в панель PocketBase и редактировать таблицы как в Excel.
